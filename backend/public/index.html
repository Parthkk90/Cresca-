🚀 AptPays Backend API

✓ Online

POST /api/close-position

Description: Close a position with multi-agent signature (server builds transaction)

Request Body:

{
  "positionId": "4",
  "userAddress": "0x..."
}

Response:

{
  "success": true,
  "rawTransaction": "hex-encoded-transaction",
  "protocolSignature": {
    "public_key": "hex",
    "signature": "hex"
  },
  "protocolAddress": "0x..."
}

📚 Documentation

This is a serverless backend API for AptPays perpetual futures protocol on Aptos blockchain.

The API provides protocol co-signing functionality for multi-agent transactions.
